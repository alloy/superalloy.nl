Guess Who’s Back

(2009-04-10)

I finally found some time to move some personal web applications to my slicehost vps, which has only been running pasternak up till now, what a waste of money. So here comes an end to that situation!

Return of the Mac and web developer

I’m back online, so step 1 has been taken care of. Now comes step 2, the hard part, keeping it up-to-date. I think I’ll be writing some shorter posts and I’d like to say “more often”, but we’ll just have to see about that. Anyways…

Thanks to Manfred Stienstra who has so kindly offered me to use his layout, because it’s just not my thing. And Webby so I can create posts locally, with my favorite text editor, and store revisions in git; welcome to superalloy.nl version 2 and hope to see you around!
